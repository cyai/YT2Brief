## Installation

To run this project locally:

1. Clone the repo
2. Run `pip install -r requirements.txt` 
3. Get API key for OpenAI
4. Update the your `.env` with your keys 
5. Run `python run.py`

We'd love you to contribute! Feel free to open PRs with improvements.
